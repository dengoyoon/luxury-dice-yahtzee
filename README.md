## 소개
- 프리미엄 고품격 주사위 게임 Yahtzee 입니다.
- 제가 만든 게임은 아니고 원래 있던 게임입니다.
- 두 명의 플레이어는 서로의 운빨을 겨룹니다.
- 운빨겜이지만 좀 더 약은 사람이 승리합니다. 이에 랭킹 시스템을 추가 할 것 입니다. (언젠가는)
- 도박으로 얻은 게임머니로 캐릭터의 옷을 구매합니다.
- 게임에서라도 부자가 되어보세요!
